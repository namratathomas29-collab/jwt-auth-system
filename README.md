# JWT Authentication System

A secure authentication system built using Java Spring Boot and JWT authentication.

## Features
- User Registration
- User Login
- JWT Token Generation
- Protected Routes
- Spring Security Integration
- H2 Database Support

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- H2 Database
- Maven

## Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- JWT Utility
- Security Configuration

## How to Run

```bash
mvn spring-boot:run
```

## API Endpoints
- POST /register
- POST /login
- GET /profile

## Purpose
This project demonstrates backend authentication and authorization using JWT tokens and Spring Security.
